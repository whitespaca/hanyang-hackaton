# 데모 스크립트

## 2분

1. 랜딩에서 “사진을 검색어로 바꾸되 AI가 결정을 대신하지 않는다”는 문제를 설명합니다.
2. 팀 자체 촬영 골판지 사진을 Web에 업로드해 Top 3/confidence를 보여줍니다.
3. 결과를 확인하고 `골판지 상자`의 테이프·송장 제거 체크리스트를 엽니다.
4. Dashboard에서 익명 분류 수가 반영됐음을 확인합니다.

## 5분

2분 흐름에 Mobile의 카메라 권한/촬영/압축, 캔 → 금속 → 캔 가이드, 최근 기록을 추가합니다. 모델 페이지에서 mock/model 상태, 원본 미저장, 지표 TBD를 설명합니다.

## 실패 대비

- 실제 모델 미로딩: `INFERENCE_MODE=mock`으로 API를 재시작하고 mock 배지를 보여줍니다.
- 실제 기기 연결 실패: Web classify를 계속 사용하고 모바일은 갤러리 UI를 시연합니다.
- 네트워크 불안정: API와 Web을 같은 노트북에서 실행합니다.
- 카메라 권한 실패: 갤러리 대안을 사용합니다.
